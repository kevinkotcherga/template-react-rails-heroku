Un template pour commencer un projet avec ruby on rails 7, react, redux et le déployer plus tard sur heroku.

Installer avec :

```
git clone 
rm -rf .git 
git init .
git add .
git commit -m "First commit"
gh repo create
git push -u origin main

npm i
```
